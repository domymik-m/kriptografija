sifriranje module¶

class sifriranje.šifrat(lokacija, tK)¶

Osnovice: object

Klasa koja reprezentira proces šifriranja fotografije u niz znakova koji se sprema u \(\text{.txt}\) datoteku.

Njezin konstruktor prima lokaciju fotografije pomoću koje se sprema slika i tajni ključ pomoću kojeg se konstruira ElGamalov ‘stroj’. U tom se slučaju kao prost broj uzima 29, što predstavlja broj znakova engleske abecede i još tri pridružena znaka: #, $ i &.
